# Recruitment Management System (ATS)

A simple Applicant Tracking System built using Python, Flask, SQLite, and Pandas. This project allows recruiters to manage candidate applications, track their status, and search/filter candidates based on skills and experience.

## Features
- **Dashboard:** View all candidates in a table format.
- **Add Candidate:** Form to input candidate details.
- **Search & Filter:** Filter candidates by Name, Skills, or Minimum Experience using Pandas.
- **Status Tracking:** Update candidate status (Applied, Shortlisted, Interview, Selected, Rejected).
- **Delete:** Remove candidate records.
- **Database:** SQLite for persistent local storage.

## Technologies Used
- **Backend:** Python, Flask
- **Database:** SQLite
- **Data Handling:** Pandas
- **Frontend:** HTML5, CSS3
